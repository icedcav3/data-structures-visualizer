# Changelog

## [1.0.0] - 2024-06-21

### Added
- Полная реализация структур данных: Stack, Queue, BST
- Интерактивная визуализация с использованием tkinter
- Графический интерфейс с панелями управления
- Функция отмены (Undo) для всех операций
- Логирование операций в JSON
- Система конфигурации пользователя
- Полное покрытие модульными и интеграционными тестами
- Docker поддержка
- Комплексная документация
- API справочник
- Руководство пользователя
- FAQ и Roadmap

### Features
- Stack операции: push, pop, peek
- Queue операции: enqueue, dequeue, front
- BST операции: insert, delete, search, traverse
- Статус-бар с сообщениями об ошибках
- Журнал операций с временными метками
- Визуальное выделение элементов
- Поддержка различных типов данных (числа, строки)

### Technical
- Архитектура MVC с разделением ответственности
- Модульный дизайн для легкого расширения
- Полная документация кода с docstrings
- Type hints для лучшей типизации
- Конфигурируемое поведение через config.json

### Testing
- 30+ тестов (unit и integration)
- >80% покрытие кода
- Тесты для всех операций
- Тесты граничных случаев
