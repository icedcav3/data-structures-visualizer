# Руководство пользователя

## Запуск приложения

```bash
python src/main.py
```

## Интерфейс

### Левая панель - Управление

1. **Выбор структуры данных**
   - Stack (Стек) - LIFO (Last In, First Out)
   - Queue (Очередь) - FIFO (First In, First Out)
   - BST (Двоичное дерево поиска) - иерархическая структура

2. **Ввод значения**
   - Поле ввода для значений
   - Поддерживает числа и строки

3. **Кнопки операций**
   - Для стека: Push (добавить), Pop (удалить), Peek (посмотреть)
   - Для очереди: Enqueue (добавить), Dequeue (удалить), Front (посмотреть)
   - Для дерева: Insert (вставить), Delete (удалить), Search (поиск), Inorder (обход)

4. **Дополнительные кнопки**
   - Clear All - очистить структуру
   - Undo - отменить последнюю операцию

### Центральная панель - Визуализация

- **Стек**: вертикальные ячейки от низа к верху
- **Очередь**: горизонтальные ячейки слева направо
- **Дерево**: иерархическое представление с кружками и линиями

### Нижняя панель - Журнал операций

- Список всех выполненных операций с временными метками
- Кнопка "Undo Last" для отмены последней операции
- Кнопка "Clear Log" для очистки журнала

### Статус-бар

Внизу окна отображаются:
- Текущее состояние ("Ready", ошибки, предупреждения)
- Результаты последней операции

## Примеры использования

### Работа со стеком

1. Выберите "Stack" в левой панели
2. Введите значение (например, 5)
3. Нажмите "Push" - значение добавляется в стек
4. Повторите для других значений (10, 15)
5. Нажмите "Pop" - удаляется верхний элемент
6. Нажмите "Peek" - видите верхний элемент без удаления

### Работа с очередью

1. Выберите "Queue" в левой панели
2. Введите значение (например, 1)
3. Нажмите "Enqueue" - добавляется в конец
4. Добавьте еще значения
5. Нажмите "Dequeue" - удаляется первый элемент
6. Нажмите "Front" - видите первый элемент

### Работа с деревом поиска

1. Выберите "BST" в левой панели
2. Введите значение (например, 50)
3. Нажмите "Insert" - создается корень дерева
4. Добавьте значения меньше (25, 12) и больше (75, 88)
5. Нажмите "Search" и введите значение для поиска
6. Нажмите "Delete" для удаления элемента
7. Нажмите "Inorder" для просмотра упорядоченного списка

## Отмена операций

Вы можете отменить последнюю операцию:
- Нажав кнопку "Undo" в левой панели
- Нажав "Undo Last" в журнале операций

Повторяющиеся нажатия отменяют операции по одной в обратном порядке.

## Советы

- Начните с малого количества элементов (5-10) для лучшей видимости
- Используйте небольшие числа (1-100) для удобства
- Смотрите журнал операций для отслеживания истории
- Экспериментируйте с разными структурами для лучшего понимания
