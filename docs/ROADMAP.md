## Известные ограничения

1. **Визуализация**
   - BST может быть перегруженным при большом количестве элементов (>100)
   - Нет автоматической масштабируемости

2. **Производительность**
   - Хранение истории может потребовать много памяти для длинных сессий
   - Нет оптимизации для больших деревьев

3. **Функциональность**
   - Нет сохранения данных между сессиями (кроме логов)
   - Нет поддержки пользовательских типов данных
