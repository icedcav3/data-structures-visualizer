# Data Structures Visualizer - Project Summary

## 📋 Описание проекта

Data Structures Visualizer - это интерактивное приложение для визуализации и изучения базовых структур данных. Позволяет пошагово выполнять операции и видеть изменения в режиме реального времени.

## ✨ Основные возможности

### Структуры данных
- **Stack (Стек)** - LIFO структура с операциями push, pop, peek
- **Queue (Очередь)** - FIFO структура с операциями enqueue, dequeue, front
- **Binary Search Tree (BST)** - иерархическая структура с insert, delete, search

### Функции
- 🎨 Интерактивная визуализация на Canvas
- 📝 Журнал операций с временными метками
- ↩️ Функция отмены (Undo) для всех операций
- 💾 Логирование операций в JSON
- ⚙️ Конфигурация пользователя
- 🧪 Полное покрытие тестами (unit и integration)
- 📦 Docker поддержка

## 🚀 Быстрый старт

### Требования
- Python 3.8+
- pip
- tkinter (обычно входит в Python)

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/icedcav3/data-structures-visualizer.git
cd data-structures-visualizer

# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python src/main.py
```

### Docker

```bash
docker-compose up
```

## 🧪 Тестирование

```bash
# Запуск всех тестов

# С отчетом о покрытии
pytest --cov=src --cov-report=html

python scripts/run_tests.py
```

## 🎯 Реализованные операции

### Stack
- ✅ push - добавить элемент
- ✅ pop - удалить верхний элемент
- ✅ peek - посмотреть верхний элемент
- ✅ is_empty - проверить пустоту
- ✅ size - получить размер

### Queue
- ✅ enqueue - добавить в конец
- ✅ dequeue - удалить с начала
- ✅ front - посмотреть первый элемент
- ✅ is_empty - проверить пустоту
- ✅ size - получить размер

### BST
- ✅ insert - вставить элемент
- ✅ delete - удалить элемент
- ✅ search - найти элемент
- ✅ inorder - обход в порядке
- ✅ preorder - обход в прямом порядке
- ✅ postorder - обход в обратном порядке
